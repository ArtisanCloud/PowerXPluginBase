## Change Log
- Feature: Prometheus `/api/v1/admin/runtime/metrics` with quota, cost, session, and restart series.
- Feature: Admin runtime APIs gated by `runtime.ops` policies (`read`, `observe`, `manage`).
- Feature: Runtime ops config surfaced in `config.yaml` with heartbeat, backoff, and observability defaults.
- Docs: Updated quickstart, runtime operations guides, and load-generator script.
- Tooling: Added `scripts/dev/quota_burst.sh` for local quota simulations.
