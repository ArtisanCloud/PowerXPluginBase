# Sample Audit Trail Bundle

This folder illustrates the artefacts generated by `make package-pxp`.

```
pxp-audit/
├── hashes.txt         # SHA256 list for every file in the package
├── audit.log          # Timestamp, source commit, staging folder
├── signature.json     # Placeholder before signing service returns real signature
└── notes.md           # Optional release notes excerpt highlighting lifecycle status
```

Use this structure when handing packages to compliance reviewers. Replace the sample files with real outputs from your release.
